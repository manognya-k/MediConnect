# MediConnect

This project is now organized as a split structure:

- `backend` - Spring Boot API (Java/Maven)
- `frontend` - UI application (currently placeholder)

## Folder Structure

```text
mediconnect/
├── backend/
│   ├── pom.xml
│   └── src/
├── frontend/
│   └── README.md
└── README.md
```

## Running Backend

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

## Running Frontend

Initialize your frontend app inside `frontend/` (for example Angular/React) and run it from there.
